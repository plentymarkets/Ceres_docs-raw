= DefaultGTCPreset

This is a preset for ShopBuilder contents. Presets can be applied during content creation to generate a default content with predefined and configured widgets. +
This particular preset generates a page for viewing the shop&#039;s general terms and conditions. It contains: +
- CodeWidget +
- SeparatorWidget +
- LegalTextsWidget
